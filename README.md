# Graph Random Walk for Catalyst Discovery

**Author:** Qinrui Liu  
**Affiliation:** Department of Materials Design and Innovation, University at Buffalo

Machine learning and graph-based random walk framework for catalyst screening and materials discovery.

This repository contains the code developed during my Ph.D. research on graph-based machine learning for catalyst discovery and materials optimization.

## Overview

This project explores the use of graph representations and random walk algorithms to identify promising catalyst candidates. The workflow combines machine learning, graph construction, and stochastic exploration to efficiently navigate the catalyst design space.

Main components include:

- Graph construction from catalyst descriptors
- Machine learning prediction of catalyst properties
- Graph-based random walk algorithms
- Network visualization and analysis
- Candidate ranking and selection
